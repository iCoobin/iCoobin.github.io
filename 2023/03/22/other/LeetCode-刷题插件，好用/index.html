LeetCode 刷题插件，好用

VS Code 安装 LeetCode 插件

  • 点击插件模块
  • 搜索 leetcode 插件
  • 点击 install 安装 leetcode 插件
  • 点击 leetcode 设置按钮，去设置

  • 选择中国区 leetcode-cn
  • 选择常用编程语言

  • command + shift + p 快捷搜索指令
  • 搜索 leetcode
  • 选择 leetcode：sign in 回车
  • 输入 leetcode 用户名或者邮箱
  • 输入 leetcode 密码
  • 登录完成
  • 点击图标切换到 leetcode 模块

  • 下面就是效果啦，所有的题库都已分类好
  • 使用起来非常方便，暴赞一下
